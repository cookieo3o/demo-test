# 💻쿠팡 고객상담관리센터💻

쿠팡 고객상담관리센터는 고객센터 상담원이 고객 정보와 상담이력을 편리하게 조회하고 관리하기 위해 사용하는 내부 웹 시스템입니다. <br>
브라우저 기반으로 동작하며 별도의 서버나 프로그램 설치 없이 실행할 수 있습니다.

---

## ✔실행 환경

- 최신 웹 브라우저
- Google Chrome 권장
- 별도의 서버 및 추가 설치 불필요


## ✔실행 방법

1. 프로젝트 폴더로 이동합니다.
2. `index.html` 파일을 실행합니다.
3. 브라우저에 고객상담관리센터 화면에 '오늘의 상담 현황'이 뜨면 정상 실행된 겁니다.


## ✔기본 사용 흐름

1. 문의 목록에서 고객을 선택합니다.
2. 선택한 고객의 상담 이력을 확인합니다.
3. 전에 비슷한 상담 이력이 있었는 지 확인합니다.
4. 상담 내용을 등록하거나 수정합니다.
5. 상담 상태를 변경하여 처리 여부를 관리합니다.


## ✔주요 기능

- 오늘의 상담 현황 조회
- 고객별 목록 조회 및 검색
- 고객별 상담 기록 조회
- 상담 내용 등록 및 수정
- 문의 상세 처리
- 상담 상태 관리
  - 접수
  - 처리중
  - 완료


## ✔이상 상황 안내

- 화면이 정상적으로 표시되지 않을 경우  
  → 브라우저를 새로고침하거나 브라우저 창을 껐다가 다시 켭니다.

- 데이터가 표시되지 않는 경우  
  → 초기 데이터 로딩 여부를 확인합니다.
